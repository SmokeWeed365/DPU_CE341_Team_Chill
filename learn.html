What did we learn from CE?


ขอแบ่งความรู้หลักๆที่ได้จากการเรียนเป็นสามอย่างนะครับ นั่นคือ
1. เครื่องมือ เช่น Visual Studio, GitHub
2. ภาษาโปรแกรม เช่น C#, Python, HTML
3. ความรู้ทั่วไป เช่น ส่วนประกอบของคอมพิวเตอร์ วิธีการทำงานCodingกับผู้อื่น

1. เครื่องมือ
โดยหลักๆในการเขียนโปรแกรมตั้งแต่ชั้นปีที่ 1 ถึง 2 เราจะใช้เป็นโปรแกรม

- Visual Studio
- Visual Studio Code

ซึ่งทั้งสองเป็นโปรแกรมที่สามารถเขียนได้กับหลายๆภาษา ซึ่งครอบคลุมภาษาที่พวกเราได้เรียนกัน นั่นคือ Python และ C#

ข้อดีของ Visual Studio และ Visual Studio Code
1. มี Extensions จากทั้ง Community และ Official ของบริษัทต่างๆที่จะทำให้การเขียน Code ของเราง่ายขึ้น
2. รับรอง Software ได้หลายภาษา รวมถึงการพัฒนา Web Service
3. สามารถ Sync กับเครื่องมืออื่นๆในการเขียน Code เช่น เชื่อมกับ GitHub Desktop ทำให้สามรถ push ไฟล์ของเราขึ้นไปยัง GitHub ได้

GitHub (GitHub Desktop)
คือโปรแกรมที่ทำให้พวกเราสามารถทำงานร่วมกันได้ โดยมี Repository เป็นตัวกลาง โดยเราสามารถ push เพื่อนำไฟล์ของเราเข้าไป หรือ pull เพื่อไฟล์ต่างๆใน Repository มาแก้ไขหรือทำต่อได้


2. ภาษาโปรแกรม
ภาษาโปรแกรม หรือ Code Language คือสิ่งที่เราจะเขียนลงไปในโปรแกรม Coding เพื่อที่จะสร้างโปรแกรมบางอย่างขึ้นมา โดยภาษาที่เราเคยเรียนนั้น จะมี
- Python เรียนการเขียนในรูปแบบของ Structured Programing (การเขียนโปรแกรมเชิงโครงสร้าง)
- C# เรียนการเขียนในรูปแบบของ Object-Oriented Programing (การเขียนโปรแกรมเชิงวัตถุ)

โดยทั้งสองภาษาจะมี Function หรือคำสั่งต่างๆคล้ายๆกัน แต่จะเขียนต่างกันเช่น
Python: print("") | C#: Console.Writeline("")


3. ความรู้ทั่วไป
การเรียนในสาขาวิศวะคอมพิวเตอร์ทำให้พวกเราได้ความรู้ต่างๆ ที่เสริมความเข้าใจเกี่ยวกับการทำงานด้านวิศวะคอมพิวเตอร์มากขึ้น
ยกตัวอย่างความรู้ทั่วไปที่ได้จากการเรียน เช่น
- ส่วนประกอบของคอมพิวเตอร์: Motherboard, CPU, GPU, RAM, Storage device
- รูปแบบการทำงานต่างๆของ Developer เช่น Agile Spiral Waterfall เป็นต้น
- การเลือกภาษาให้เหมาะสมกับรูปแบบงาน เช่น การเขียนโปรแกรมแบบ OOP เหมาะกับ ภาษา C#


The way we want to pursue a career


ในแนวทางของการเรียนวิศวะคอมพิวเตอร์ พวกเรามีตัวเลือกมากมายในการประกอบอาชีพ
สุขหทัย และ เทพประทาน อยากที่จะเป็น Game Developer
จอมปราชญ์ อยากที่จะเป็น System Analyst
และ ปิยธิดา อยากที่จะเป็น Business Analyst

Game Developer : เป็นผู้อยู่เบื้องหลังการผลิตเกมออกสู่ตลาด โดยมีหน้าที่สร้างและพัฒนาระบบเกมในส่วนของการเขียนโปรแกรมและการเขียนโค้ด ต่างๆ
System Analyst : ทำหน้าที่วิเคราะห์ปัญหา และนำมาเปลี่ยนให้เป็นรูปที่เหมาะสมสำหรับการประเมินค่าด้วยเครื่องคอมพิวเตอร์ รวมถึงการวางแผนงาน การสั่ง การทบทวนโครงการประมวลผลโดยคอมพิวเตอร์
Business Analyst : เป็นตำแหน่งที่ช่วยในการกำหนดปัญหาทางธุรกิจโดยจะวิเคราะห์จากข้อมูลเชิงลึก สถิติทางเทคนิค ต่างๆ เมื่อเข้าใจปัญหาอย่างชัดเจนแล้ว นักวิเคราะห์จะร่างข้อกำหนดโดยละเอียดสำหรับวิธีการแก้ปัญหา
